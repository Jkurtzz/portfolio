To-Do List Full-Stack Web Application

Purpose:

The purpose of this project is to learn and demonstrate the necessary skills of a full-stack engineer.

Overview:

This project was done to demonstrate the necessary skills of a full-stack engineer like setting up a database, creating a front-end user interface, and connecting the two with an api.
User authentication was implemented with hashed passwords, sessions, and Json Web Tokens.

Languages:

This app uses a MySQL database.
The api was written using Express.js.
The front-end was written using React.js.

Security and Authentication:

The app uses three levels of authentication. Using the Node.js crypto library, the user's information was hashed using the sha512 hashing algorithm. Passport-Local was used to serialize and deserialize users into sessions. Finally, Passport-JWT was used to sign and validate Json Web Tokens with an RSA key pair.

Publication:

The app was published using Amazon Web Services (AWS). I purchased a domain name and received an SSL/TLS certificate for it. My app was then hosted on an EC2 instance and an Application Load Balancer connected the domain to the instance.
